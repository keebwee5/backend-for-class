What's flask?

Flask is a lightweight, micro web framework for Python, designed to make it easy to build simple web applications quickly. Flask provides tools and libraries for routing, templating, and handling requests, but it leaves out a lot of other functionality, giving developers the flexibility to choose their preferred tools or add only whatâ€™s needed. This simplicity and modularity make Flask ideal for small to medium projects or for building APIs.
